# 📝 To-Do App (Flutter)

## 1. Introduction
This is a simple and efficient To-Do application built using Flutter. The app helps users manage their daily tasks by allowing them to add, update, and delete tasks in an easy-to-use interface.

---

## 2. Technologies Used
- Flutter
- Dart
- Material UI (Flutter widgets)
- VS Code / Android Studio

---

## 3. Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Real-time UI updates using setState
- Simple and clean user interface

## 5. Development Process
- Designed the UI using Flutter widgets such as Scaffold, Column, and ListView  
- Implemented task management using a List  
- Used StatefulWidget to manage dynamic data  
- Applied setState to update the UI when tasks are added or removed  
- Tested the app on emulator/device for responsiveness
